# Checkpoint - Desenvolvimento Back-End PHP

Repositório com as atividades de checkpoint da Carreira PHP da Alura.

## Estrutura

- `Checkpoint 1`: projeto Laravel com Composer/PSR-4, MVC, MySQL, domínio de produtos, CRUD web, autenticação, autorização, validação, CSRF, rate limiting, API REST v1 e documentação OpenAPI/Swagger.

O `Checkpoint 1` possui um `README.md` próprio com instruções de configuração, execução, testes, rotas web e API.
